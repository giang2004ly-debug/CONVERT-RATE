# CONVERT-RATE
"CONVERT  RATE"  là một hệ thống cho phép người dùng chuyển đổi tiền tệ giữa các quốc gia khác nhau dựa trên tỷ giá tương ứng. Người dùng có thể lựa chọn loại tiền, nhập số tiền cần chuyển đổi và xem kết quả nhanh chóng. Hệ thống hỗ trợ quản lý và kiểm tra dữ liệu tỷ giá, giúp đảm bảo kết quả chuyển đổi chính xác và thuận tiện cho người sử dụng.
